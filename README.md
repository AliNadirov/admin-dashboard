# Admin Dashboard

A modern and responsive admin dashboard built with React and Vite.  
This project demonstrates real-world front-end patterns such as API integration, reusable components, state management, search, pagination, settings persistence, and dark mode support.

---

## Overview

This project was developed as a front-end dashboard application focused on building a realistic admin panel experience.

It includes dynamic statistics, searchable data tables, responsive layouts, reusable UI components, and persistent user settings.

---

## Features

- Dynamic dashboard statistics based on API data
- Category-based chart for product insights
- Users page with real-time search and pagination
- Products page with real-time search and pagination
- Global navbar search connected to page content
- Settings page with saved preferences using localStorage
- Dark mode support
- Skeleton loading states
- Responsive layout for desktop, tablet, and mobile

---

## What This Project Demonstrates

- Building a scalable React application
- Structuring reusable components and layouts
- Managing state with React hooks
- Handling loading, error, and empty states
- Working with external API data
- Persisting settings with localStorage
- Creating responsive dashboard interfaces

---

## Tech Stack

Frontend: React, Vite  
Routing: React Router  
HTTP Requests: Axios  
Styling: CSS3  
Charts: Recharts  
Version Control: Git & GitHub
